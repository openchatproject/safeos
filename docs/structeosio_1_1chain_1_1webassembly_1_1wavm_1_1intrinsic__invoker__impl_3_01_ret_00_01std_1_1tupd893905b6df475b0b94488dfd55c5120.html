Detailed Description

template<typename Ret, typename... Inputs, typename ... Translated>
struct eosio::chain::webassembly::wavm::intrinsic_invoker_impl< Ret, std::tuple< const name &, Inputs... >, std::tuple< Translated... > >

Specialization for transcribing a reference to a name which can be passed as a native value This type transcribes into a native type which is loaded by value into a variable on the stack and then passed by reference to the intrinsic.

Template Parameters
Ret- the return type of the native method
Inputs- the remaining native parameters to transcribe
Translated- the list of transcribed wasm parameters
