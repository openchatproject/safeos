struct eosio::chain::webassembly::binaryen::intrinsic_invoker_impl< Ret, std::tuple< null_terminated_ptr, Inputs... > >

Specialization for transcribing a null_terminated_ptr type in the native method signature This type transcribes 1 wasm parameters: a char pointer which is validated to contain a null value before the end of the allocated memory.
