template<typename T, typename Ret, typename... Inputs, typename ... Translated>
struct eosio::chain::webassembly::wavm::intrinsic_invoker_impl< Ret, std::tuple< T *, Inputs... >, std::tuple< Translated... > >

Specialization for transcribing a pointer type in the native method signature This type transcribes into an int32 pointer checks the validity of that memory range before dispatching to the native method

Template Parameters
Ret- the return type of the native method
Inputs- the remaining native parameters to transcribe
