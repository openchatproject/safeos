Detailed Description

template<typename Ret, typename... Inputs>
struct eosio::chain::webassembly::binaryen::intrinsic_invoker_impl< Ret, std::tuple< const fc::time_point_sec &, Inputs... > >

Specialization for transcribing a reference to a fc::time_point_sec which can be passed as a native value This type transcribes into a native type which is loaded by value into a variable on the stack and then passed by reference to the intrinsic.

Template Parameters
Ret- the return type of the native method
Inputs- the remaining native parameters to transcribe
